## API Overview

```cpp
// Load latency matrix from CSV
LatencyMatrix matrix = LatencyMatrix::from_csv("latency.csv");
matrix.symmetrize();  // Optional: average L[i][j] and L[j][i]

// Build topology with default config
TopologyBuilder builder;
TopologyGraph graph = builder.build(matrix);

// Or load config from JSON
TopologyBuilder builder = TopologyBuilder::from_config("config.json");
TopologyGraph graph = builder.build(matrix);

// Export results
std::string adj_list = graph.to_adjacency_list();
std::string dot = graph.to_dot();
std::string json = graph.to_json();
```

## Algorithm Summary

1. Parse CSV -> LatencyMatrix (NxN symmetric matrix)
2. Build Dendrogram using single-linkage hierarchical clustering
3. Cut dendrogram at tier thresholds (default: 5us, 15us, 50us)
4. Assign tiers to physical nodes based on clustering
5. Infer hidden nodes (switches) for each cluster
6. Construct edges connecting physical nodes to hidden nodes
7. Export graph in desired format
